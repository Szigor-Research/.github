## Boundaries

Public software is provided for technical research and education. It is not a
custody service, broker, payment processor, investment adviser, or promise of
production readiness. Review repository-specific security and limitation
documents before use.

Private production data, credentials, models, routing, signing, incident
records, and operator telemetry are never part of the public reference set.
